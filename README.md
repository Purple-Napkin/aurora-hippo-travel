# Hippo Travel

Travel / experiences-oriented Aurora storefront template (same commerce core as Hippo Grocery; customize copy, schema, and pages for your domain). Uses [`aurora-starter-core`](../aurora-starter-core).

- **Studio:** template **Hippo Travel** (`hippo-travel`).
- **GitHub:** `Purple-Napkin/aurora-hippo-travel`.

See [Hippo storefront docs](../aurora-studio/docs/hippo-storefront-templates.md).

```bash
pnpm install && cp .env.example .env.local && pnpm dev
```
